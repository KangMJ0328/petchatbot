# 나만없어펫 — 챗봇 시나리오

## 1. 챗봇 이름
나만없어펫

## 2. 챗봇 주요 기능
카카오톡 그룹 채팅방에서 친구들과 함께 가상 펫을 키우는 소셜 육성 게임

### 핵심 기능
- 먹이주기 (7종 먹이, 진화 분기에 영향)
- 훈련 (근력/지능/매력 스탯 성장)
- 진화 시스템 (알 → 아기 → 어린이 → 청소년 → 성체, 3가지 분기)
- 탐험 (1~4시간 방치형 보상)
- 랭킹 (전체 서버 펫 순위)

### 소셜 기능
- 공동 양육 (같은 방 멤버가 함께 펫 키움)
- 기여도 랭킹 (방 내 누가 가장 많이 기여했는지)
- 심부름 (다른 채팅방 펫과 교류)
- 약탈/방어 (랜덤 이벤트, 협동 방어)

### 미니게임
- 가위바위보 (골드 미니게임)
- 돌발 퀴즈 (60초 타임어택)
- 운세 (일일 1회)

## 3. 대표 시나리오

### 시나리오 1: 펫 생성 & 먹이주기
```
유저: @나만없어펫 시작
봇: [BasicCard] 🥚 펫이 태어났어요! (알 이미지)
    버튼: [먹이주기] [펫 정보] [상점]

유저: @나만없어펫 먹이
봇: [TextCard] 🛒 먹이를 선택하세요! (보유: 100G)
    🍞 빵 — 10G
    🍖 고기 — 20G
    🥩 스테이크 — 35G
    버튼: [빵] [고기] [스테이크]

유저: @나만없어펫 먹이 고기
봇: [BasicCard] 🍖 고기를 먹였어요! (펫 이미지)
    🍖 포만도: 70% / ✨ 경험치: 18/22
```

## 4. 사용 말풍선

| 컴포넌트 | 용도 |
|---------|------|
| SimpleText | 일반 텍스트 응답 |
| TextCard | 텍스트 + 버튼 (명령어 선택) |
| BasicCard | 이미지 + 텍스트 + 버튼 (펫 상태) |

- QuickReplies 미사용
- buttonLayout: vertical
- 응답은 간결하게 (채팅창 절반 이하)

## 5. 타겟 유저

- 연령: 전연령 (10대~30대 메인)
- 그룹: 친구/동료 카카오톡 그룹 채팅방
- 사용 빈도: 일 5~10회 (먹이, 훈련, 출석 등)
