# 카카오 그룹챗봇 입점 가이드 정리

**출처**: 카카오 제공 Google Slides
**정리일**: 2026-04-09

---

## 1. 그룹 챗봇이란

- 그룹 챗봇은 **팀채팅/일반채팅**에서만 동작 (오픈채팅, 나와의채팅 미지원)
- 그룹 챗봇은 **제휴 업체에 한해 권한**을 받은 후 서비스 가능
- 채널 1개에 **그룹 챗봇 + 채널 1:1 챗봇** 둘 다 연결 가능
- 운영채널: 실 서비스에 연결하여 노출할 채널
- 개발채널: 챗봇 개발 목적으로 연결하는 채널
- **운영채널과 개발채널은 서로 다른 채널을 사용해야 함**

---

## 2. 챗봇 시나리오 전달

카카오에 시나리오 문서를 전달해야 함. 전달 양식 (자유형식):

1. **챗봇 이름**: 나만없어펫
2. **챗봇 주요 기능**: 펫 키우기 게임
   - 먹이주기, 훈련, 진화
   - 탐험, 퀴즈, 운세
   - 랭킹, 심부름
3. **대표 시나리오**: 챗봇 응답 스크린샷

> 챗봇 응답만으로 사용자에게 충분한 정보를 제공해야 함 (외부 웹 랜딩 지양)

---

## 3. 사업자 기능 제한 (개인인 경우)

사업자가 아니라도 챗봇 서비스는 가능하지만 제한 있음:

### 사용 제한 API
- 카카오 디벨로퍼스에서 앱 연결 시 사업자번호 필수
- 앱 연결 없으면 일부 API 제한:
  - 채널, 카카오싱크 관련 API
  - **이벤트 API (선톡 발송)도 제한됨**

### 채널 프로필 경고 문구
- "사업자 정보가 확인되지 않은 채널" 경고 배지 표시
- 제거하려면 chatbot@kakaocorp.com으로 문의

---

## 4. 프로필 가이드

### 이름 규칙
- ❌ 일반적이고 보편적인 이름 지양 (예: 게임봇, 건강봇, 이미지봇)
- ✅ 브랜드 아이덴티티 반영 (예: 나만없어펫)

### 프로필 이미지 규칙
- ❌ 이미지 내 글자 사용 지양
- ❌ 리스크 키워드/이미지 지양 (의료, 금융 등)

### 전달 방법
- 이름 + 프로필 이미지 후보를 chatbot@kakaocorp.com으로 전달

---

## 5. 채널 홈 고객센터 경로

- 채널 홈에 **고객 문의 루트** 안내 필요 (이메일, 전화번호 등)
- 채널 홈 "기본 정보"에 이메일/홈페이지 설정
- 채널 1:1 채팅방도 CS 루트로 활용 가능

---

## 6. 채널 연결하기

### ① 카카오 비즈니스 가입
- https://business.kakao.com 접속
- 카카오톡에서 사용하는 계정으로 로그인
- **상태: ✅ 완료**

### ② 카카오톡 채널 만들기
- 채널명 = 챗봇 프로필로 사용됨
- 채널명 규칙:
  1. 어떤 기능/서비스인지 직관적으로 이해 가능
  2. 타이핑하기 쉽고 간결 (대소문자, 띄어쓰기 X)
- 프로필 이미지 내 텍스트 사용 지양
- **상태: ✅ "나만없어펫" 완료**

### ③ 비즈니스 채널 인증
- 사업자인 경우: 사업자등록번호 입력
- **개인인 경우: "아직 사업자 등록을 하지 않으셨나요?" 클릭 → 건너뛰기**
- 경고 배지 표시됨 (수용 또는 카카오에 제거 문의)
- **상태: ⚠️ 개인 → 건너뛰기**

### ④ 개발용 채널 별도 생성 (필수!)
- **개인**: 일반 채널 2개 생성 (운영 1 + 개발 1)
- **사업자**: 개발자용 채널 만들기 가능
- 개발 채널로 연결된 봇은 **테스트용 팀채팅방 1개에서만** 사용 가능
- **상태: ❌ 해야 함**

### ⑤ 채널 공개 설정 On
- 채널 정보 → 옵션 설정 → 채널 공개 On
- 공개해야 카카오톡에서 노출됨
- **상태: 확인 필요**

### ⑥ 카카오싱크/디벨로퍼스 연결
- **사업자만 가능** → 개인은 건너뛰기
- 카카오 디벨로퍼스에 앱 등록 → 채널 연결
- **상태: 해당 없음 (개인)**

## 7. 권한 받기

### ① 챗봇 관리자센터 로그인
- https://chatbot.kakao.com 접속
- 카카오톡 계정으로 로그인

### ② 카카오 계정 전달
- 카카오 비즈니스 사이트 + 챗봇 관리자센터 로그인 후 해당 계정 전달
- 마스터 권한: 1명만 (카카오톡 가입 계정)
- 권한 받으면 "그룹 챗봇" 탭 + "그룹 챗봇 만들기" 버튼 생김

## 8. 챗봇 제작하기

### ① 채널 연결
- 챗봇 관리자센터 > 설정 > 챗봇 관리 > 카카오톡 채널 연결
- **개발채널과 운영채널은 반드시 다른 채널 사용**
- 개발 채널 = 배포 없이 수정사항 바로 반영

### ② 봇 한줄 설명
- 팀채팅 챗봇 목록에 노출
- **14자 이내** 권장 (줄바꿈 방지)
- 우리 설명: `친구들과 함께 펫을 키워요!`

### ③ 도움말 블록
- 시나리오 > 도움말 블록 > 봇 응답에 도움말 버튼
- 챗봇 목록에서 "도움말" 버튼 클릭 시 표시
- 채팅방에서 `@봇이름 도움말` 발화 시 표시

### ④ 웰컴 메시지 (봇 입장 블록)
- 채팅방에 챗봇 추가 시 자동 발송
- **내용은 짧게** (채팅창 절반 이하)
- 하단에 '챗봇 멘션하기' 버튼 → `@챗봇이름` 자동 입력

### ⑤ 대표 명령어 설정 (자동완성)
- 시나리오 설정 > 대표 명령어 설정
- `@봇이름` 멘션 시 또는 `/` 입력 시 명령어 자동 노출
- 가나다 초성순 → 최근 사용순으로 변경
- 파라미터 필요 시 "버튼 사용 설명"으로 안내
- 반영까지 최대 1시간 소요

## 9. 챗봇 개발 및 테스트

### 개발 가이드
- 1단계: 기본 가이드 (공통)
- 2단계: 그룹챗봇 가이드 (→ KAKAO_DEV_GUIDE.md 참고)

### 사용 가능한 말풍선
- 텍스트, 이미지, 리스트, 리스트(랭킹), 피드형(ItemCard)
- ❌ QuickReplies, CommerceCard, Carousel 미지원

### 플러그인
- 시간입력, 공유, 이미지 플러그인 사용 가능

### 멘션
- 말풍선 1개에 최대 15명
- 말풍선 3개로 최대 45명
- 나간 멤버 멘션 불가

### 테스트 방법
1. 챗봇 마스터가 **방장인 팀채팅** 생성
2. 개발 채널에 봇 추가 (`@봇이름`으로 호출)
3. 개발 봇은 **최대 1개 팀채팅방**에서만 사용 가능
4. 테스트 완료 후 팀채팅 초대링크를 chatbot@kakaocorp.com으로 전달

### 챗봇 오픈 시
- 채널 홈 > 카드 추가 > "챗봇 초대" 카드 추가 > 토글 ON

### ⚠️ 주의사항
- 채널 연결 해제 시 → 모든 팀채팅방에서 봇 강제 퇴장
- 마스터 변경 시 → 기존 개발 팀채팅방에서 봇 퇴장
- 개발채널과 운영채널은 **반드시 다른 채널**

## 11. 유의사항 (필수 준수)

### 광고
- 챗봇 응답에 광고 포함 불가
- 랜딩 페이지 광고 시 카카오 사전 검토 필요

### AI/데이터
- 유저 발화 AI 학습 활용 금지
- 발화 저장 시 도움말에 저장 사실 + 기간 명시

### 선톡 (이벤트 API)
- Beta 기간 무상, 종료 시 사전 공지
- 월 최대 2회, 심사 필요
- 금지: 광고, 홍보, 유해, 대량 퇴장 유발

### 응답
- 채팅창 전체 가리면 안 됨 (짧고 간결하게)
- "결제" 구매 유도 금지
- 광고성/유해성 문구 금지
- "대화내용 열람" 등 오인 표현 금지

### 대표 명령어
- 직관적이면 설명 생략
- 파라미터 필요 시만 설명 추가
